A computable, self-checking database

Because the formulas are stored too, it can

derive · reconcile · invert

run the formulas forward, cross-check redundant routes, and (next) solve them backwards. Not a data lake.

Open and forkable

Storing science the way git stores code.

A semantic layer for scientific AI

Agents act on typed physics, not text. An invalid step is a type error, not a silent wrong number. Redundant routes are a built-in verifier, so verification scales with compute.
