openmaterials.ai

A database for physics.

The schema is the physics itself: a graph of typed quantities and the formulas that relate them. Append your simulations and measured values from papers; the database derives, reconciles, and keeps everyone's results honest. Clone it, fork it, host it. Storing science the way git stores code.

Variables and formulas

Every node is a typed physical quantity; every edge is its symbolic equation. The order of operations is the map.

Simulations and experiments

Values from codes and from papers attach to the same variables as projections, and reconcile at the observables.

Open and forkable

The whole database is files in a repo. git clone to run locally; open a pull request to append.
